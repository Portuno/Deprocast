Deprocast — Personal Productivity Command Center

This is a static, no‑JavaScript page intended for crawlers and text‑only tools (e.g., ChatGPT browsing) to understand the product.

Value Proposition

Deprocast helps you conquer procrastination with a neuroscience‑informed workflow: AI‑generated micro‑tasks, a guided “Next Task” module, a structured journal, and a project‑centric dashboard.

Key Features

  • Projects: select and manage projects with metadata (category, difficulty, target date).
  • AI Micro‑tasks: generate micro‑tasks per project via an assistant (Mabot).
  • Dashboard: metrics for completion, in‑progress, and pending tasks.
  • Next Task: prioritized recommendation and quick start CTA.
  • Journal: entries with mood, energy, focus, tags, and key events.
  • Calendar & Protocols: planning and behavioral playbooks.
  • Auth: email/password and Google (Supabase).

Navigation Map

  • Landing: value prop + CTA to sign in.
  • App (/app): protected area with TopBar, sidebar/bottom nav, and content panes.
  • Dashboard: stats + generation + next task + journal + task list.
  • Projects / Journal / Calendar / Protocols / Profile pages.

Tech Snapshot

  • React + Vite + Tailwind.
  • Supabase: Auth, DB, Edge Functions (Mabot integration).
  • Deployed on Vercel as SPA; route fallback to /index.html.

How to Review

For a deeper review, log in to the app and inspect:

  • Dashboard coherence (metrics vs. task states).
  • Micro‑task quality and acceptance criteria.
  • Journal structure and usefulness for retrospectives.
  • Project metadata impact on AI generation.
